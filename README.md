# ToDo_FastAPI

Обычный ToDoList, сделанный для того, чтобы показать работу с библиотекой FastAPI и немного Pydantic
Так же данный мини-проект демонстрирует работу с обработкой и записью в файлы json формата, в отдельной ветке от main планируется сделать это все с интерфейсом

## Заметки
- ✅ Проект закончен на 75% (Всегда есть, что добавить и к чему стремиться :)) , версия с интерфейсом возможно не будет реализована, так что лучше не ждать

## Функции
- 📋 Просмотр всех задач
- 📝 Отметка выполненых задач
- ✅ Добавление новых задач
- 🗑️ Удаление ненужных задач
- 🎯 Фильтрация по выполненным/не выполненным задачам

## Установка и запуск

Вы можете скачать данный проект как с готовым файлом 'tasks.json', так и удалить этот файл после скачивания, программа сама его создаст пустым, после чего вы сможете добавлять туда свои задачи.

Чтобы запустить программу, надо открыть код в одной из IDE установить необходимые заивисимости (FastAPI, Pydantic) и после нажать кнопку запуск в файле main.py, в терминале появится ссылка, как только вы ее нажмете откроется браузер и в адресной строке добавьте '/docs', откроется автоматическая документация от FastAPI, где будут показаны все доступные функции.
